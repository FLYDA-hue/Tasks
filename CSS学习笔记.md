## 学习内容

### CSS与HTML的关系

##### HTML定义整体框架和基本结构，而CSS是在此基础上进行样式的选择和设计。

### CSS语法

##### css通常由选择器、属性和属性值组成，多个规则可以进行组合。

##### 1.选择器的声明中可以写多条属性。

##### 2.每一行属性都要以英文分号结尾。

##### 3.所有属性和属性值都要以键值对形式出现。

### CSS三种导入方式

##### 1.内联样式（Inline Styles）（HTML的style中）。

##### 2.内部样式表（Internal Stylesheet）（HTML的head中来定义）。

##### 3.外部样式表（External Stylesheet）(用链接link可以多个网页使用相同样式)。

##### 优先级1>2>3

### 选择器

###### 选择器是CSS中的关键部分，允许针对特定元素定义样式。

- ##### 元素选择器

  元素/标签{元素声明

  }

- ##### 类选择器

  .类名{元素声明

  }

- ##### ID选择器

  #id名{元素声明

  }

- ##### 通用选择器（对所有的元素进行一个选择）

  *{元素声明

  }

- ##### 子元素选择器（选择直接位于这个父元素内部的子元素）

​	.父元素元素名>.子元素元素名{元素声明

​	}

- ##### 后代选择器/包含选择器

​	.父元素元素名 与子元素并列div的内部元素{元素声明

​	}

- ##### 并集选择器/兄弟选择器

  元素1+元素2{元素声明

  }

  ###### 注意：元素1和元素2并列。

- ##### 伪类选择器（给用户交互文档结构，鼠标悬停时的效果）

​	元素：hover{元素声明

​	}

### CSS属性

###### 属性有上百个（可以在菜鸟教程、w3school在线教程上查看）。

##### color（字体颜色）、background-color（背景颜色）、font-size（字体大小）、font-family（设置字体）、font-weight（字体粗细）、line-height（调整行距）

###### 复合属性：通过一个属性设置多个样式。

##### font（设置字体样式）、padding、border、margin

### 盒子模型

###### 是CSS常用于布局的基本概念，每个元素都可以被看作是一个盒子，包含Content（内容）、Height和Width（内边距）、Border（文本边框）、Margin（外边距）。

|      属性名       | 说明                                                      |
| :---------------: | :-------------------------------------------------------- |
|  内容（Content）  | 盒子包含的实际内容如文本、图片等。                        |
| 内边距（Padding） | 围绕在内容的内部，文本与边框间的距离，用padding属性设置。 |
|  边框（Border）   | 围绕在内边距外部，是盒子的边界，用border属性设置。        |
| 外边距（Margin）  | 围绕在边框外部，用margin属性设置。                        |

- ##### 边框(border):solid(实线)、dashed(虚线)、dotted(点线)、double(双实线)。

  ###### border-width遵循上右下左。

### 传统网页布局方式

###### 本质上就是摆盒子，网页布局中常多种布局方式参杂在一起。

- ##### 标准流（普通流、文档流）：网页按照元素书写顺序依次排序

- ##### 浮动

- ##### 定位

- ##### Flexbox和Grid（自适应布局，可以在手机端访问也可以在电脑端访问）

### 浮动

###### 元素脱离文档流,可以改变元素默认的排列顺序，根据开发者意愿可以在网页任意方向布局。

##### 浮动属性用来创建浮动框，将其移动到边缘触及包含块或另一浮动框边缘。（与行内块的区别：浮动是紧挨着的，而行内块彼此之间是有缝隙的）

##### 选择器{

##### 	float:left/right/none;

##### }

###### 浮动相对于父元素浮动，仅在其内部浮动。

##### 三大特性：脱离标准流控制；多个盒子一行内显示、顶部对齐排列；具备行内块元素特性。

##### 清除浮动：在父元素里加overflow：hidden；添加伪元素.父元素：：after{content：""

##### 			display:table;

##### 			clear:both；

##### }

### 定位

###### 定位布局可以精准定位，但缺乏灵活性。

##### 定位方式：

- ##### 相对定位：相对于正常位置进行定位（position：relative）。

- ##### 绝对位置：相对于其最近的已定位父级元素进行定位，不占据文档流（position：absolute）。

- ##### 固定定位：相对于浏览器窗口进行定位，不占据文档流，固定不随滚动而移动（position：fixed）。
